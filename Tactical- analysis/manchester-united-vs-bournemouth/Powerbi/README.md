## Overview
This Power BI dashboard visualises key performance metrics from the Manchester United vs Bournemouth Premier League match, building on the baseline framework established in the Chelsea vs Everton analysis.

The dashboard is designed to communicate **both performance output and match dynamics** in a clear, coach-friendly format.

---

## Included Metrics

### Baseline Metrics (Consistent with Match 1)
- Total shots by team
- Total expected goals (xG)
- Shot outcome breakdown
- Shot distance distribution

These metrics provide a consistent reference point across matches.

---

## Additional Metrics Introduced (Match 2 Progression)

### Shot Timing Analysis
- Shots grouped into defined match time windows
- Highlights periods of sustained pressure and momentum shifts

### Goal Event Timeline
- Chronological listing of goal events
- Includes minute, team, player, and xG value

### Efficiency Indicator
- xG per shot used as a proxy for chance quality

---

## Data Sources
- FBref (numerical match and event data)
- Understat (shot location and xG timeline visuals)
